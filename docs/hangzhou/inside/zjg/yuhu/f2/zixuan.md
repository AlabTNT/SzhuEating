# 玉湖餐厅自选区

!!! tip "信息卡"
    **地址**：西区玉湖餐厅二楼东侧  
    **参考价格**：7~20￥  
    **“一块六”所在地**：待补充
    **回收处**：餐厅南侧与风味区连接处  
    **平均人流量**：中

    > 午餐 11:00-13:00  
    > 晚餐 16:30-18:30  

## 总体评价

主供中式菜品，属于生命体征餐，7块钱就能吃饱，如果你一定要吃早餐且能接受每次上完课都通勤回玉湖吃两餐，那么一天吃饭开销不会超过20块钱。

如果你是游客，那么不建议你体验本食堂。
