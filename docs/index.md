# TNT Eating

你好!  
欢迎来到TNT Eating网站! 这里是AlabTNT在技术力低下&无心push Turing Eating后，自己独立门户使用私有服务器搭载的网站，目的在于记录并分享TNT去过的饭店（食堂）及品尝过的菜品。  

起初，TNT Eating目标与Turing Eating一致，希望创造一个开放的、以浙大图灵班为中心的Eating Share社区，但苦于各种原因，TNT最终决定选用管理制，只允许已授权的人参与进改进TNT Eating，且放大范围至全国，开辟更多美食绿洲。  
后面，TNT还在考虑要不要将TNT Eating开发至TNT Playing，记录一些可以休闲娱乐的场所，但限制于精力与时间，这项计划短时间内不可能完成。
